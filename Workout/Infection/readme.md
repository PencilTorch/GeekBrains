Написать программу, моделирующую процесс распространения
инфекционного заболевания на местности размером
клеток (задается с клавиатуры, массив динамический), если:
-первая зараженная клетка задается с клавиатуры (их количество
и положения на рабочем поле);
-в каждый интервал времени инфекционная клетка может с
вероятностью 0,5 заражать каждую из соседних здоровых клеток
(слева, сверху, справа и слева);
-в каждый интервал времени инфекционная клетка может с
вероятностью 0,25 заражать каждую из соседних здоровых клеток
по диагонали (слева сверху, слева снизу, справа сверху, справа
снизу);
-после пяти единиц времени зараженная клетка не воспринимает
инфекцию, а полученный иммунитет сохраняется во время
следующих четырех единиц времени, после чего клетка становится
доступной к инфицированию.
Вывести на экране текущее состояние заражения после каждого
интервала времени с учетом зараженных, здоровых и тех
клеток, получивших иммунитет.
